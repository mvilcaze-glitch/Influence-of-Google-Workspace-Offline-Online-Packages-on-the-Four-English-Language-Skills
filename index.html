My First Web Page

Educational Portal: Influence of Google Workspace Offline-Online Packages on the Four English Language Skills

Student Researcher: María de los Ángeles Vilca Zeballos • Actividad 2

Program Purpose

This web prototype has been designed to model and systematize the pedagogical processes of the educational intervention program at Antigravity IDE, Arequipa (2026).

Using semantic HTML5 markup, this framework demonstrates how offline-enabled Google Workspace components boost listening, reading, speaking, and writing competencies in high school settings.

Learning Objectives

  1. Identify and implement structural HTML5 tags to design semantic, accessible layouts.
  2. Incorporate interactive client-side JavaScript and Web Audio API to simulate educational tools.
  3. Evaluate localized data persistence configurations to assist learning in connection-intermittent classrooms.
